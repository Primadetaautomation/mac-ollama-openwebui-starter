# Security Policy

## Supported

Dit is een starterskit zonder runtime code op de host. Beveiliging focust op:
- Up-to-date container images (pull regelmatig)
- Beperkte netwerkblootstelling (standaard alleen localhost)
- Geen secrets in de repo; gebruik environment variables/host secrets

## Vulnerabilities melden

Open een privé-issue of neem contact op met de repo maintainers. Voeg reproduceerbare stappen toe en, indien mogelijk, een voorstel voor mitigatie.
